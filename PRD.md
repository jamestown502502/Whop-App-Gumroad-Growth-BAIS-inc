# PRD — Gumroad Growth Hub

**Product:** Gumroad Growth Hub  
**Builder:** Bennett AI Solutions Inc.  
**Platform:** Whop App Store  
**Version:** 1.0 MVP

## One-Line Description
A Whop app that gives Gumroad sellers curated resources, tools, and downloads to grow their business.

## Target User
Gumroad sellers who want structured guidance, tools, and templates — delivered inside their Whop workspace.

## Problem
Gumroad creators often lack a centralized place for growth resources, actionable tools, and ready-to-use templates. They waste time hunting across blogs and YouTube.

## MVP Features
- Member home dashboard with stats and featured resources
- Full resource library (categorized, linked)
- Creator tools directory (free tools only for MVP)
- Downloads section (PDF templates and checklists)
- Seller dashboard (Whop B2B view)

## Tech Stack
- Next.js 15 + App Router
- @whop/sdk, @whop/react
- Tailwind CSS
- Vercel (deploy)
- No database required for MVP (static content)

## Monetization
- Whop-native billing (no Stripe needed for MVP)
- App listed on Whop App Store → drives inbound
- BAIS brand visibility inside every member view

## Success Criteria for This Session
- All routes working and rendering correctly
- Deployed to Vercel with working live URL
- Whop dashboard paths configured
- Passes Whop App Store submission checklist
- Full buyer/member flow works end-to-end
