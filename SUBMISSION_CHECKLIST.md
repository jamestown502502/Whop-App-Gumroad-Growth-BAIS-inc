# Whop App Store Submission Checklist

## Required Before Submitting to Discover

- [ ] App has a clear name: **Gumroad Growth Hub**
- [ ] Short description written (see APP_STORE_LISTING.md)
- [ ] Full description written (see APP_STORE_LISTING.md)
- [ ] App icon created (512x512 PNG, clean design)
- [ ] 10–20 second demo video recorded
- [ ] 2–3 screenshots captured (1280x800 recommended)
- [ ] Category selected: **Education** or **Productivity**
- [ ] App preview URL tested: https://whop.com/apps/<app_id>
- [ ] Both experience and dashboard paths work end-to-end
- [ ] No broken links in-app
- [ ] BAIS branding visible ("Built by Bennett AI Solutions Inc.")
- [ ] App reviewed in Whop dev proxy (localhost test passed)

## Rejection Risks to Avoid
- Broken routes or 404s on experience or dashboard paths
- Missing icon or placeholder description
- No real content (we have static content — this is satisfied)
- App path not set in dashboard (must be exact: `/experiences/[experienceId]`)
